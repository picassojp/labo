Detalle sobre la competencia final:
Dentro de esta carpeta se encuentran todos los scripts utilizados para cada modelo y los BO_log de cada HT. 
Los mejores modelos según la optimización bayesiana fueron el KF1, KF2 y KF7. Para estos se utilizó el script
z1291 (basado en el script original del compañero Tomas Delvecchio) y los resultados del semillerío de cada modelo
se guardo en la carpeta "semillerio_modelos". En esa misma carpeta se encuentra el archivo excel donde se juntaron
las predicciones de cada semillerío y se realizó una hibridación mediante votación. Así se obtuvo el archivo .csv
que finalmente se subío a la última competencia.

Además, cada uno de los experimentos se fue registrando en el archivo "registro_experimentos.xlsx".
